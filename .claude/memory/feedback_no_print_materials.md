# 印刷物・物理成果物は不要

本田康志様の seiho-ai-study プロジェクトでは、**印刷物・物理成果物（QRコード印刷、配布資料、ポスター等）は前提にしない**。

## Why
- 当日の参加者アクセスはスマホで Web を直接開く想定（プロジェクター投影・口頭案内で URL/QR を共有）
- 紙物は本田様の運用ポリシー外（明示要求がない限り想定しない）
- 過去：E1 タスクに「QRコード印刷物準備」を独断追加していたが、実態と齟齬があった（2026-06-26 にユーザー指摘で修正）

## How to apply
- **追加しない**：研修・配布系のタスクで「印刷物」「PDF 配布物」「ポスター」「名刺サイズQR」等を勝手に提案しない
- **executor 越権の回避**：物理成果物の必要性は decision-maker（本田様）の判断。明示要求があったときのみ着手
- **既存記述の削除**：設計書・タスク・コンテンツ内に「印刷物」前提の記述があれば指摘・削除する
- **Web 完結の運用**を基本前提に：QR は Web に表示する SVG（既実装）で十分、印刷ではなく投影 / スクリーン共有を想定
